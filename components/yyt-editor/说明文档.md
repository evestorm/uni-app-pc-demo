## yyt-editor(富文本编辑器)

##### 作者:于大明
### 使用方法

```javascript
<yyt-editor ref="yytEditor" :inputText="copyItem.studentMeRemark"></yyt-editor>
this.$refs.yytEditor.getText(); // 获取文本
```
